Elementary Cellular Automata

This program displays all 256 rules of Wolfram's Elementary Cellular Automata on a 2D canvas. Each individual Cellular Automaton is based on a 1-dimensional array(grid) filled with cells. A cell has two possible states: 0 (white) or 1 (black).

Each 1-dimensional array represents a generation (starting from 0 instead of 1) within the 2D canvas. To determine the state of the next generation of cells the current we look at 3 cells: the current generation cell and its two neighbors (cells to the left and right). The rule number when converted to a 8 bit binary number will show the 8 possible state configurations of the next generation cell. The rule number and the size of each cell can both be altered.
